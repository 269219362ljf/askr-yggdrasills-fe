# askr-yggdrasills-fe

``` 
npm start   
npm test   
npm run build   
npm run eject   

```

yarn下可用的命令   

``` 
yarn start   
yarn test   
yarn build   
yarn eject   
```

docker使用，指定docker容器3000端口映射到docker容器端口3000

```
docker build -t askr-yggdrasills-fe .
docker run -p 3000:3000 askr-yggdrasills-fe
```
